x-client-info 参数

参数名称 是否必须 参数类型 参数描述
device_type 是 string 目前只接受app.ios,app.android,web.pc,web.h5,3rd.wechat,3rd.qq
device_id 是 string 设备的唯一标示,非app则使用生成的uuid
device_brand 是 string app:手机品牌 pc:pc品牌,获取不到默认值pc
app_channel 否 string 渠道
app_version app必填 string app版本号
package_name app必填 string 包名
package_type app必填 string debug / alpha / beta / release
system_version 是 string 手机/pc 操作系统版本及类型，如iOS 12.4,windows 10等
system_language 是 string 系统语言
browser_version 非app必填 string 浏览器版本,如果是三方软件(例如微信)内嵌浏览器，则为三方软件版本
network_type app必填 string 当前网络类型:Wi-Fi 5G / 4G / 3G / 2G
referer_key 否 string 调用来源唯一标示
idfa 否 string iOS平台广告标示符
  • app必填：客户端调用的情况下必填，当device_type为app.*客户端调用的情况下必填
  • 非app必填：非客户端调用情况必填
