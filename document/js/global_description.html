接口响应

接口返回格式统一为

{
    "code": 0,
    "message": "success",
    "internal": false,
    "data": {
    }
}
  • http status: 200 和 500，除内部错误为500外，其它均为200，业务错误码通过code标识
  • code： 错误码标识，只有为0表示成功
  • message:简要说明，如code为0时值为success
  • data：接口返回具体的数据，部分接口data为空
  • internal:在code!=0时出现，用于标识错误，有且为false时，message的信息可以直接展示，否则不应直接展示message内容
