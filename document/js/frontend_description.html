全局请求header

header参数名称 是否必须 参数类型 参数简介
x-api-key 是 string 会员SaaS系统分配给接入方的 api key，用于识别接入方
x-api-secret 是 string 会员SaaS系统分配给接入方的 api secret ，用于权限校验
Origin 是 string 分配的域名，用来校验是否为安全配置域名 例如: http://www.google.com
Authorization 需要登录验证的接口为必须 string 参数值为登录请求接口中获取到的 access-token 的值，注意要在获取到的字符串最前面加上Bearer
accept-language 是 string 传入语言

Signature 参数

  • Signature jwt加密秘钥

    网关会分配给接入方一个secret，这个secret， 由网关统一管理分发 ，此secret用来加密Signaturre参数jwt字符串。

  • Signature jwt payload 详情

    在请求时候需要通过秘钥添加入jwt生成相应的 Signature 字符串添加到请求header中， 然后在 JWT 的 payload 字段中添加如下终端信息参数，此参数每次接口请求时候都必须带上，可以访问 jwt.io 加密&验证。

参数名称 是否必须 参数类型 参数描述
iss 是 string 此参数为固定值 member-account-system-kit
exp 是 int 过期时间，为当前uninx时间戳+7200秒，例：1545726078
alg 是 string 加密方式 ，固定值：HS256
x-client-info 是 合并x-client-info内容
